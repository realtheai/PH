#### 라이선스

- **사용 가능 여부**: ⚠️ **연구용으로 명시, 대회 사용 전 확인 필요**

#### 데이터 샘플
```csv

```

#### 피싱 유형 (14가지)
1. 공공기관 사칭 계좌이체 유도 (328건)
2. 인터넷뱅킹·카드론·예금 편취 (232건)
3. 사고및보상문의 (175건)
4. 상품가입및해지 (175건)
5. 이체출금대출서비스 (175건)
6. 잔고및거래내역 (175건)
7. 기타 8가지 유형
