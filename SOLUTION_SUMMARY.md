# OpenAI Vision OCR 문제 해결 보고서

## 📋 **문제 상황**

OpenAI Vision API가 일부 피싱/스미싱 이미지에서 텍스트 추출을 거부하는 현상 발생:
- "죄송하지만, 해당 이미지의 텍스트를 추출할 수 없습니다."
- "I'm unable to extract text from images directly."

## 🔍 **원인 분석**

### 1. 이미지 특성 분석 결과:
- ❌ 파일 크기: 성공 3.0KB vs 실패 3.7KB (큰 차이 없음)
- ❌ 해상도: 성공 148×130 vs 실패 155×97 (유사)
- ❌ 포맷: JPEG/PNG 둘 다 성공/실패 케이스 존재
- ✅ **결론**: 이미지 기술적 특성이 원인이 아님

### 2. 실제 원인:
**OpenAI의 콘텐츠 안전 필터가 프롬프트에 반응**
- 같은 이미지에 대해 프롬프트에 따라 다른 응답
- "보안 분석", "중립적" 프롬프트 → 거부율 높음
- "OCR 전용", "연구 목적" 프롬프트 → 성공률 높음

## ✅ **해결책**

### 최종 채택 프롬프트:
```python
"Perform OCR on this image. Extract all text in original language (Korean/English). Output text only."
```

### 개선 사항:
| 항목 | 기존 | 개선 후 |
|------|------|---------|
| 프롬프트 언어 | 한국어 (긴 설명) | 영어 (간결) |
| 온도(Temperature) | 0.2 | 0.1 |
| 성공률 | ~60% | ~80% |

- ✅ 성공: 4개 (80%)
- ❌ 실패: 1개 (20%)
